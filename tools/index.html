Upload network file

This uploads the "network" file exported from wigle. The server then go through the data and update/add information to the database.

Upload location file

This uploads the "location" file exported from wigle. The server then go through the data and update/add information to the database.

Upload client file

This uploads the .csv files from airodump. The server then go through the data and update/add information to the database. (aircrack-ng version 1.2-rc2 and later).

Upload old client file

For .csv files made by: aircrack-ng version 1.2-rc1 - Released 31 October 2014. Replaced 10 April 2015. Earlier versions might also work here.

Update database

Update fields in network table

This sets: channel, band, lastseen, icon, hidden ssid. And overwrites best lat/lon with last lat/lon if blank.

Update fields in location table

This sets month_seen. And delete position with corrupt/missing location data.

Update connected clients in network table

Gets the MAC from connected clients and puts it in network table

Update probing clients in network table

Gets the MAC from probing clients and puts it in network table

Lookup MAC address for routers

Checks each router's MAC address against macvendors.com and populates the network table with correct vendor

Lookup MAC address for clients

Checks each client's MAC address against macvendors.com and populates the clients table with correct vendor
